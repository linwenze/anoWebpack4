### 1、安装

```
git clone https://github.com/linwenze/anoWebpack4.git
cd anoWebpack4
npm install
```
### 2、运行
```
npm run start-dev //开发环境
npm run start-test //测试环境
npm run start-prod //生产环境
```
### 3、打包
```
npm run start-dev //开发环境
npm run start-test //测试环境
npm run start-prod //生产环境
```
- 开发环境打包生成后放在根文件下的dev文件夹
- 测试环境打包生成后放在根文件下的dist文件夹
- 生产环境打包生成后放在根文件下的prod文件夹
